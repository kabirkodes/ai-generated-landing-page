# FlowDesk - AI Generated Landing Page

A responsive one-page marketing website built with HTML5 and CSS3.

## What I built
- Hero section with headline and CTA button
- Features section with 3 cards using CSS Grid
- Call-to-action section
- Footer with links
- Fully responsive with media queries

## AI Assistance
AI suggested the app name, hero copy, feature cards, and colour palette. All HTML and CSS was written manually.
